Detaily implementace
--------------------
Použitá verze Javy: Java 11
Možnosti build.xml:
    Při spuštění ant compile je stažena knihovna JavaFX, pokud již není ve složce ./lib.
    Při spuštění ant compile je vygenerována dokumentace.
    Při spuštění ant run je spuštěn i příkaz ant compile, jestliže není k dispozici výsledný JAR archiv.
    Při spuštění ant clean se smažou všechny vytvořené soubory (build, doc, dest).
    Při spuštění ant doc je vygenerována dokumentace.
